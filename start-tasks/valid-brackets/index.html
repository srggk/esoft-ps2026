📚 Проверка правильной последовательности скобок

Алгоритм проверяет, является ли строка скобок () [] {} корректной. Стек-решение, сложность O(n).

📌 Примеры для тестирования:

() → true ✅
()[]{} → true ✅
(] → false ❌
([)] → false ❌
{[]} → true ✅
{[()]} → true ✅
((( → false ❌
] → false ❌
(пусто) → true ✅
